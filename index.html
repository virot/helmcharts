Welcome

This repository hosts Helm charts for easy deployment of applications on Kubernetes clusters. Helm is a package manager for Kubernetes that simplifies the deployment and management of applications.

Available Charts

versitygw

A Helm chart for deploying versitygw

v0.1.0

Getting Started

Add this Helm repository:

helm repo add virot https://virot.github.io/helmcharts/

Update your Helm repositories:

helm repo update

Install a chart:

helm install my-release virot/versitygw

Requirements

  • Kubernetes cluster (v1.19+)
  • Helm 3.x installed
  • kubectl configured to access your cluster
