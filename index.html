Helm Charts Repository

Welcome to the Helm charts repository. This repository hosts Helm charts that can be used to deploy applications on Kubernetes.

Getting Started

To use the charts from this repository, you need to add it to your Helm client:

helm repo add virot https://virot.github.io/helmcharts/
helm repo update

Available Charts

  • versitygw - A Helm chart for deploying versitygw

Installing a Chart

To install a chart from this repository:

helm install my-release virot/versitygw

To see all available versions and configuration options:

helm search repo virot
helm show values virot/versitygw
